# Microservices starting point

set-up ```.env``` file.

```env
MONGO_INITDB_DATABASE=[databaseName]
MONGO_INITDB_ROOT_USERNAME=[usernameForDatabase]
MONGO_INITDB_ROOT_PASSWORD=[passwordForDatabase]
```
